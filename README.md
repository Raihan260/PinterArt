# PinterArt Vite + React

Project React dengan Vite, siap dipakai tanpa halaman khusus.

## Quick Start (Windows PowerShell)

```powershell
cd "d:\Semester 5\PinterArt_MMS\pinterart-vite-react"
npm install
npm run dev
```

- `npm run build`: Build untuk produksi.
- `npm run preview`: Preview build produksi.
- `npm run lint`: Cek linting.
- `npm run format`: Format dengan Prettier.

## Struktur
- `src/` berisi entry `main.jsx` dan komponen `App.jsx` minimal.
- `vite.config.js` mengaktifkan plugin React.
- `index.html` memount app ke `#root`.
